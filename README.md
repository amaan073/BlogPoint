# Static Blog Website (Practice)

A static multi-page blog website built using HTML, Bootstrap, and vanilla JavaScript.  
This project was created as a **practice exercise** to improve front-end layout, navigation, and basic DOM manipulation skills.

---

## What This Project Covers

- Multi-page website structure
- Navigation using relative HTML links
- Responsive layout with Bootstrap
- Basic JavaScript interactivity
  - Active navigation states
  - Like button toggle
  - Comment section using DOM manipulation
- Simple contact form handling

---

## Pages

- Home
- About
- Contact
- Individual blog post pages (static content)

---

## Tech Stack

- HTML
- CSS
- Bootstrap
- JavaScript (vanilla)

---

## Purpose

This is a **learning project** built to practice:
- structuring a static website
- maintaining consistent layouts across pages
- adding basic interactive behavior with JavaScript

The blog content is static and hard-coded.

---

## Notes

- No backend or database
- No dynamic data or authentication
- Layout duplication is expected for a static site

---

## Status

Completed as a practice project.
